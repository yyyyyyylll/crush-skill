# 聊天截图分析指引

通过 Claude 的视觉能力分析聊天截图，提取消息内容。

使用方式：在 Claude Code 中直接用 Read 工具读取截图图片，然后按照本指引处理。

---

## Step 1: 识别平台

- 微信：绿色/白色气泡，头像在两侧
- iMessage：蓝色/灰色气泡
- QQ：类似微信但 UI 细节不同
- Telegram：蓝色/白色气泡，圆角更大
- 其他：记录平台特征

## Step 2: 提取消息

对每条可见消息提取：
- 发送者（左侧 = 对方/crush，右侧 = 用户）
- 消息内容（文字、表情、图片描述）
- 时间戳（如可见）
- 消息类型（文字/语音/图片/表情包/红包/链接）

## Step 3: 标注细节

- 已读标记（如可见）
- 消息间隔（根据时间戳或对话上下文推断）
- 表情包内容描述
- 语音条长度（如可见）

## Step 4: 输出格式

```json
{
  "platform": "wechat",
  "messages": [
    {
      "sender": "crush",
      "content": "消息内容",
      "timestamp": "14:30",
      "type": "text",
      "emojis": ["😂"]
    }
  ],
  "observations": [
    "crush的消息比用户的长",
    "crush用了3个emoji"
  ]
}
```

## Step 5: 快速洞察

提取完成后，直接给出初步观察：
- 消息长度对比
- 谁更主动（话题发起者）
- emoji 使用情况
- 整体对话氛围（热络/平淡/冷淡）
